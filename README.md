# BdayBasti Quiz

Lokales Quizprojekt mit:

- Team-Erstellung vor dem Start
- Quiztabelle mit 5 Kategorien und 5 Punktestufen
- Team-Punktestand rechts mit `+100` und `-100`
- Frageansicht mit gestufter Aufdeckung
- Speicherung aller Daten in `localStorage`
- Reset-Knopf, der alles loescht und zur Team-Erstellung zurueckbringt

## Starten

Die App braucht keinen Build-Schritt. Einfach `index.html` im Browser oeffnen.

## Struktur

- `index.html` baut die Seite auf
- `styles.css` enthaelt das Layout und Styling
- `app.js` steuert Navigation, Punktestand und lokalen Speicher
